# TechStore Project

## 📦 Описание

Это простой frontend-проект, собранный с использованием Vite.
Проект представляет собой верстку интернет-магазина.

---

## 🚀 Запуск проекта

### 1. Клонировать репозиторий

```
git clone <ссылка-на-репозиторий>
```

### 2. Перейти в папку проекта

```
cd techstore
```

### 3. Установить зависимости

```
npm install
```

### 4. Запустить проект

```
npm run dev
```

После этого проект будет доступен по адресу:

```
http://localhost:5173
```

---

## 📦 Сборка проекта

Для создания production-сборки:

```
npm run build
```

Собранный проект будет находиться в папке:

```
/dist
```

---

## 🔍 Предпросмотр сборки

```
npm run preview
```
